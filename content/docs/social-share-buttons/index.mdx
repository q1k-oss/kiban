### When To Use

Use **SocialShareButton** when you need to:

- Share content across multiple social media platforms
- Render a group of platform-specific share buttons from a single component
- Maintain consistent UI and behavior across applications
- Abstract platform-specific share URLs and logic
- Easily enable or disable supported platforms
- Reuse social sharing functionality across consumer apps

`SocialShareButton` centralizes social media sharing logic inside the UI Kit, allowing consumer applications to focus only on providing the content to be shared.

---

### Features

- Supports multiple platforms (LinkedIn, Reddit, Twitter/X, Threads, etc.)
- Reusable and configurable button group
- Platform-specific share URL handling
- Consistent styling via UI Kit
- Easy to consume and extend
- Future-ready for additional share actions (Copy link, Email, etc.)
