## When To Use

- Display non-blocking notifications or feedback to users
- Show success, error, warning, or info messages
- Provide feedback after user actions with optional undo/action buttons
- Show countdown timers with pause/resume on actionable toasts

## API

### kibanToast

Simple toast notifications with variant styling.

```tsx
import { kibanToast } from "@q1k-oss/kiban";

kibanToast.success("Saved!");
kibanToast.error("Something went wrong", { description: "Try again." });
kibanToast.warning("Storage low", { duration: 8000 });
kibanToast.info("New update available");
```

| Method | Description |
|--------|-------------|
| `kibanToast.success(title, options?)` | Green success toast |
| `kibanToast.error(title, options?)` | Red error toast |
| `kibanToast.warning(title, options?)` | Yellow warning toast |
| `kibanToast.info(title, options?)` | Blue info toast |
| `kibanToast.loading(title, options?)` | Spinner toast (duration: Infinity by default) |
| `kibanToast.promise(promise, options)` | Loading → success/error based on promise result |
| `kibanToast.custom(render, options?)` | Render fully custom JSX toast |
| `kibanToast.dismiss(id)` | Dismiss a toast by ID |

**Options:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `description` | `string` | — | Secondary text below the title |
| `duration` | `number` | `5000` | Auto-dismiss duration in ms |
| `position` | `"top-left" \| "top-right" \| "bottom-left" \| "bottom-right" \| "top-center" \| "bottom-center"` | `"top-right"` | Position of this specific toast |

### actionableToast

Rich toast with progress bar, countdown timer, and action button.

```tsx
import { actionableToast } from "@q1k-oss/kiban";

actionableToast({
  title: "Item deleted",
  description: "The item has been removed.",
  variant: "error",
  duration: 8000,
  action: {
    label: "Undo",
    onClick: () => console.log("Undone!"),
  },
});
```

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | — | Toast title (required) |
| `description` | `string` | — | Secondary text |
| `variant` | `"success" \| "error" \| "warning" \| "info"` | `"success"` | Color variant |
| `duration` | `number` | `5000` | Countdown duration in ms |
| `action` | `{ label: string; onClick: () => void }` | — | Action button |

### Toaster

Place `<Toaster />` once in your app layout. All props from sonner's `Toaster` are supported.

```tsx
import { Toaster } from "@q1k-oss/kiban";

export default function Layout({ children }) {
  return (
    <>
      {children}
      <Toaster position="top-right" />
    </>
  );
}
```

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `position` | `"top-left" \| "top-right" \| "bottom-left" \| "bottom-right" \| "top-center" \| "bottom-center"` | `"top-right"` | Where toasts appear on screen |
| `closeButton` | `boolean` | `true` | Show close button on default sonner toasts |
| `duration` | `number` | `5000` | Default auto-dismiss duration in ms |
