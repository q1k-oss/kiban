## When To Use

- Feedback after user actions (save, delete, submit)
- Async operation status (loading, success, error)
- System notifications that don't block the UI
- Confirmations with undo/action buttons

## API

### kibanToast

```tsx
import { kibanToast } from "@q1k-oss/kiban";

kibanToast.success("Saved!", { description: "All changes persisted." });
kibanToast.error("Failed", { description: "Try again.", position: "bottom-right" });
kibanToast.loading("Processing...");
kibanToast.promise(fetchData(), {
  loading: "Fetching...",
  success: "Done!",
  error: "Failed.",
});
kibanToast.custom((id) => <MyComponent onClose={() => kibanToast.dismiss(id)} />);
kibanToast.dismiss(toastId);
```

| Method | Description |
|--------|-------------|
| `success(title, options?)` | Success toast |
| `error(title, options?)` | Error toast |
| `warning(title, options?)` | Warning toast |
| `info(title, options?)` | Info toast |
| `loading(title, options?)` | Spinner toast (stays until dismissed) |
| `promise(promise, options)` | Loading → success or error |
| `custom(render, options?)` | Custom JSX layout |
| `dismiss(id)` | Dismiss by ID |

**Options (KibanToastOptions):**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `description` | `string` | — | Secondary text |
| `duration` | `number` | `5000` | Auto-dismiss in ms |
| `position` | `KibanToastPosition` | `"top-right"` | Toast position |
| `showProgress` | `boolean` | `true` | Show countdown bar |
| `colors` | `ToastColors` | — | Override variant colors |
| `className` | `string` | — | Additional CSS classes |

### kibanActionableToast

```tsx
import { kibanActionableToast } from "@q1k-oss/kiban";

kibanActionableToast({
  title: "Delete item?",
  variant: "error",
  duration: 10000,
  actions: [
    { label: "Delete", icon: "trash-2", onClick: async () => { await deleteItem(); } },
    { label: "Cancel", icon: "x", onClick: () => {} },
  ],
});
```

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | — | Toast title |
| `description` | `string` | — | Secondary text |
| `variant` | `"success" \| "error" \| "warning" \| "info"` | `"success"` | Color variant |
| `duration` | `number` | `5000` | Countdown duration in ms |
| `showProgress` | `boolean` | `true` | Show countdown bar and pause/resume |
| `colors` | `ToastColors` | — | Override variant colors |
| `className` | `string` | — | Additional CSS classes |
| `action` | `ActionableToastAction` | — | Single action button |
| `actions` | `ActionableToastAction[]` | Yes / No | Multiple action buttons |

**ActionableToastAction:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `"OK"` | Button text |
| `icon` | `string` | `"check"` | Lucide icon name |
| `onClick` | `() => void \| Promise<void>` | no-op | Click handler (async shows spinner) |
| `variant` | `"default" \| "outline" \| "ghost"` | `"outline"` | Button style |

### ToastColors

Override any variant color per-toast.

| Prop | Type | Description |
|------|------|-------------|
| `iconColor` | `string` | Icon color |
| `borderColor` | `string` | Solid border color |
| `borderGradient` | `string` | Gradient border background |
| `bgColor` | `string` | Toast background |
| `progressColor` | `string` | Progress bar color/gradient |

### Toaster

Place once in your app layout.

```tsx
import { Toaster } from "@q1k-oss/kiban";

<Toaster position="top-right" />
```
