## When To Use

- When you need to collect a phone number from a user
- When you need to validate a phone number
- When you need to format a phone number
- When you need to display a phone number in a specific format
- When you need to internationalize a phone number
- When you need to localize a phone number
- When you need to format a phone number in a specific format
- When you need to internationalize a phone number
- When you need to localize a phone number
