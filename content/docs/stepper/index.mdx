## When To Use

- When you need to represent progress across multiple steps
- When a process is divided into multiple stages (e.g. onboarding, verification)
- When you want users to complete tasks in a guided flow
- When you need visual feedback for completed, current and upcoming steps
- Useful for forms, wizards, authentication flows, and multi-screen processes

## Accessibility

- Uses clearly visible active and completed states for accessibility
- Can be navigated programmatically using step indexes
- Labels for steps improve readability for assistive technologies
