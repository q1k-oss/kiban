## When To Use

- When you need a panel that slides in from the edge of the viewport
- For side navigation, filter panels, or configuration drawers
- To display detail views or forms without navigating away
- When you need a secondary adjacent panel for drill-down content

## Props

### SheetContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"right"` | Which edge the sheet slides from |
| `positioning` | `"fixed" \| "absolute"` | `"fixed"` | Fixed overlays the viewport. Absolute positions within the nearest relative parent. |
| `showOverlay` | `boolean` | `true` | Show/hide the dark backdrop overlay |
| `preventOutsideClose` | `boolean` | `true` | Prevent closing when clicking outside the sheet |
| `className` | `string` | — | Additional classes |

### SheetAdjacent

A secondary panel that renders adjacent to the main sheet. Place inside `SheetContent`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | `false` | Whether the adjacent panel is visible |
| `side` | `"left" \| "right"` | `"left"` | Which side of the main sheet the panel appears |
| `width` | `string` | `"w-full"` | Tailwind width class for the panel |
| `gap` | `number` | `0` | Gap in pixels between main and adjacent panel |
| `title` | `string` | — | Title text rendered in the adjacent panel header |
| `onClose` | `() => void` | — | Callback to close the adjacent panel. Renders an X button in the header. |
| `className` | `string` | — | Additional classes |

### SheetHeader

Renders a flex row with children on the left and a close button on the right. Accepts `SheetTitle`, `SheetDescription`, or any element (like a `Button`) as children.

### SheetTitle

Renders the sheet's title text.

### SheetDescription

Renders secondary description text below the title.

### SheetFooter

A flex container for action buttons at the bottom.

## Accessibility

- Uses `role="dialog"` with `aria-labelledby` and `aria-describedby`
- Focus is trapped within the sheet when open
- `Escape` key closes the sheet
- Overlay click closes modal sheets (unless `preventOutsideClose` is true)
