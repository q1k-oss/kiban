### When To Use

- When you need a panel that slides in from the edge of the viewport
- For creating side navigation menus, filter panels, or configuration panels
- To display additional information without navigating away from the current view
- When you need a mobile-friendly alternative to modal dialogs

### Props

#### Sheet

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `behavior` | `"modal" \| "panel"` | `"modal"` | Modal shows overlay and traps focus. Panel renders inline without overlay. |
| `open` | `boolean` | â€” | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | â€” | Callback when open state changes |

#### SheetContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"right"` | Which edge the sheet slides from |
| `positioning` | `"fixed" \| "absolute"` | `"fixed"` | Fixed overlays the viewport. Absolute positions within the nearest relative parent. |
| `className` | `string` | â€” | Additional classes |

#### SheetHeader

Renders a flex row with children on the left and a close button on the right. Place `SheetTitle`, `SheetDescription`, or any element (like a `Button`) as children.

#### SheetTitle

Renders the sheet's title text.

#### SheetDescription

Renders secondary description text below the title.

#### SheetFooter

A flex container for action buttons at the bottom.

### Accessibility

- Uses `role="dialog"` with `aria-labelledby` and `aria-describedby`
- Focus is trapped within the sheet when open
- `Escape` key closes the sheet
- Overlay click closes modal sheets (disabled for panel behavior)
