### When To Use

Use **TextEditor** when you need to:

- Provide a modern rich text editing experience similar to Notion or Google Docs  
- Allow users to format text with bold, italic, underline, strikethrough, and code  
- Support structured content like headings, lists, quotes, and code blocks  
- Enable image embedding via URLs or local uploads  
- Customize text appearance with colors, highlights, and font sizes  
- Build document-style editors with floating menus and toolbars  

---

## Features

- **Two Editor Variants** – Agent Editor (fixed toolbar) and Notion Editor (floating & bubble menus)  
- **Inline Formatting** – Bold, italic, underline, strikethrough, code, links, highlights  
- **Font Customization** – Custom font sizes, text colors, and highlight palettes  
- **Rich Content Blocks** – Headings (H1–H6), lists, blockquotes, code blocks, dividers  
- **Image Support** – Insert images from URLs or local device uploads  
- **Interactive Links** – Add and style hyperlinks  
- **Text Alignment** – Left, center, and right alignment  
- **Undo / Redo** – Full history support  
- **Custom Toolbars** – Replace default menus with your own components  
- **Heading Anchors** – Auto-generated IDs and anchor links for headings

## API Reference

### TextEditor Props

| Prop | Type | Default | Description |
|-----|------|---------|-------------|
| `value` | `string` | `'<h1>Untitled Document</h1>'` | Initial HTML content |
| `onChange` | `(content: EditorContent) => void` | — | Fired on content change (html, json, text) |
| `variant` | `'AGENT_EDITOR' \| 'NOTION_EDITOR'` | `'AGENT_EDITOR'` | Editor UI variant |
| `placeholder` | `string` | `"start typing..."` | Placeholder text |
| `editable` | `boolean` | `true` | Enable or disable editing |
| `wrapperClassName` | `string` | — | Wrapper container classes |
| `editorClassName` | `string` | — | Editor content classes |
| `headingLevels` | `number[]` | `[1,2,3]` | Supported heading levels |
| `enableHeadingAnchors` | `boolean` | `false` | Auto-generate heading IDs |
| `fontSizes` | `FontSizeOption[]` | Default sizes | Custom font sizes |
| `colors` | `string[]` | Default palette | Text color options |
| `highlightColors` | `HighlightColor[]` | Default highlights | Highlight color options |
| `topToolbar` | `(editor) => ReactNode` | — | Custom top toolbar (Agent Editor) |
| `bubbleMenu` | `(editor) => ReactNode` | — | Custom bubble menu (Notion Editor) |
| `floatingMenu` | `(editor) => ReactNode` | — | Custom floating menu (Notion Editor) |
| `onImageUpload` | `(file: File) => Promise<ImageUploadResult>` | — | Async image upload handler for drag & drop, paste, and toolbar |
